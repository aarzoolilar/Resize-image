# Image Resizer Tool

This project resizes and converts images in batch using Python and Pillow.

## Features
- Reads all images from a folder
- Resizes images automatically
- Converts images to PNG format
- Saves them in a new folder

## Technologies Used
- Python
- Pillow Library
- OS module

## How to Run

1. Install Pillow
pip install pillow

2. Add images inside the "images" folder.

3. Run the script
python image_resizer.py

4. Resized images will appear in the "resized_images" folder.
